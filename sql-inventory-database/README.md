# SQL Inventory Database

This project is a complete SQL database system for inventory management.

## Features
- Products, suppliers, and purchase transactions  
- Stored procedures  
- Triggers  
- Joins and advanced queries  
- MySQL ready  

## Structure
```
schema/
queries/
procedures/
triggers/
README.md
```

## How to Use
1. Run `create_tables.sql`
2. Run `insert_data.sql`
3. Execute queries, procedures, and triggers as needed
